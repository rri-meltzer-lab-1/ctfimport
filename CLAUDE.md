# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

This is a fork of the EEGLAB plugin `ctfimport` (originally from sccn), used at Rotman Research Institute. It imports CTF MEG/EEG data into EEGLAB (MATLAB).

## Version Control

Use `jj` (Jujutsu) for all version control operations — not `git` directly. The repo has a `.jj/` directory colocated with `.git/`.

## Commit Messages

Follow the Conventional Commits standard (e.g., `feat:`, `fix:`, `docs:`, `refactor:`).
